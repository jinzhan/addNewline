# fx [命令]

## 命令：
  fx newl  Add newline at the end of file

## 选项：
```
--help        显示帮助信息                                              [布尔]
--version     显示版本号                                                [布尔]
--suffix, -s  The suffix of files(split with commas)   [字符串] [默认值: "js"]
--test, -t    Show the target files
```
